Certified Kubernetes Application Developer(CKAD) - Exam Tips

Finally completed my CKAD certification few weeks back.This is my first open book exam. Definitely, a new experience. This exam tries to test your hands on skills about Kubernetes. This certificate was intended for Developers who will be using kubernetes in their day-to-day activities.

Tips to clear exam

  1. CKAD requires you to solve 19 questions in 2 hours . Approximately 6 mins per question. You will see questions with different weightage with Debugging type and creating new resources type.
  2. Check the context on each question. Four different clusters will be used. There are certain questions which needs to be executed in particular node. Read the question carefully.Commands to change the context are provided on top of the question itself. Make sure to copy and execute before answering the questions.
  3. Try to attend more weightage question first. Questions on State Persistence,Pod Design,Observability will have high weightage and easy scoring options.
  4. You can access the nodes as root user if needed. Instructions to do so, are given in the exam guide and will be available to access anytime during exam.
  5. Try to use imperative commands throughout the exam to save time. Click this link to get imperative commands
  6. Udemy course by Mumshad Mannambeth on CKAD was very helpful to clear this exam. Also give lab exercises to have hands on. It covers all the topic for CKAD exam.
  7. Practice more and hands on is very important. Try to allocate only 6 mins for each question.
  8. Please check this link to get all the commands used.Personally recommend to download minishift and practice in your local machine.

Finally

We have one retry attempt for this exam. So even if you fail in first attempt, don't worry it will be good learning experience to understand the exam pattern and you can focus on things which you need more practice. Happy Learning !!!!!
