Java8 Features - BiConsumer Functional Interface

In this tutorial, We are going to learn about BiConsumer Functional Interface which was introduced as part of Java 8. As learned in previous tutorial about Functional Interface, we have only one abstract method in Predicate Function Interface called accept(Object,Object).This method will accept inputs and doesn't return any value.

Methods in Consumer

MethodName type Description
accept(T t,U u) abstract Process the given input data and return nothing.
andThen(BiConsumer<? super T,? super U> after) default Chain multiple BiConsumer function and execute one after the other.

Example and explanation

  • accept(T t,U u)

T,U are the inputs given to the BiConsumer function. In below example , we are just multiplying two values.
BiConsumer multiplyConsumer = (a, b) -> {
System.out.println("Multiply -->" + a * b);
};
multiplyConsumer.accept(5,3);

  • andThen(BiConsumer<? super T,? super U> after)

This method takes another BiConsumer Function as input and execute one consumer after other. In the below example,we are multiplying and adding two numbers in two different functions. 
BiConsumer multiplyConsumer = (a, b) -> {
System.out.println("Multiply -->" + a * b);
};

BiConsumer addConsumer = (a, b) -> {
System.out.println("Add -->" + (a + b));
};

addConsumer.andThen(multiplyConsumer).accept(5, 3);

Complete Code For Consumer

	
package com.weblearner.functionallambda;

import java.util.function.BiConsumer;

public class BiConsumerFunctionalInterface {

	public static void main(String args[]) {

		BiConsumer multiplyConsumer = (a, b) -> {
			System.out.println("Multiply -->" + a * b);
		};

		BiConsumer addConsumer = (a, b) -> {
			System.out.println("Add -->" + (a + b));
		};

		addConsumer.andThen(multiplyConsumer).accept(5, 3);

	}

}
