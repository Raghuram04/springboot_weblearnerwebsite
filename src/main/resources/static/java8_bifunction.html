Java8 Features - BiFunction Interface

In this tutorial, We are going to learn about BiFunction Functional Interface which was introduced as part of Java 8. As learned in previous tutorial about Functional Interface, we have only one abstract method in Predicate Function Interface called apply(Object, Object).

Methods in Function

MethodName type Description
apply(Object, Object) abstract Apply the function to give inputs
andThen(Function<? super R,? extends V> after) default Chain multiple function together. Apply the first BiFunction to given input and second Function on the evaluated response.

Example and explanation

  • apply(T t,U u)

Interface BiFunction<T ,U ,R> where T,U are input parameter and R is the return type.In the below example, we are concating two string together. 
BiFunction fullNameFunction = (firstName, lastName) -> firstName + " " + lastName;
fullNameFunction.apply("John", "Peter");

  • andThen(Function<? super R,? extends V> after)

BiFunction fullNameFunction = (firstName, lastName) -> firstName + " " + lastName;
Function lengthFunction = (fullName) -> fullName.length();	
fullNameFunction.andThen(lengthFunction).apply("John", "Peter");

Complete Code For Function

	
package com.weblearner.functionallambda;

import java.util.function.BiFunction;
import java.util.function.Function;

public class BiFunctionFunctionInterface {

	public static void main(String[] args) {

		BiFunction fullNameFunction = (firstName, lastName) -> firstName + " " + lastName;
		Function lengthFunction = (fullName) -> fullName.length();

		System.out.println("FullName -> " + fullNameFunction.apply("John", "Peter"));
		System.out.println("Length Of FullName -> " + fullNameFunction.andThen(lengthFunction).apply("John", "Peter"));

	}

}
