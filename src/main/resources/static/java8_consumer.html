Java8 Features - Consumer Functional Interface

In this tutorial, We are going to learn about Consumer Functional Interface which was introduced as part of Java 8. As learned in previous tutorial about Functional Interface, we have only one abstract method in Predicate Function Interface called accept(Object).This method will accept a input and doesnt return any value.

Methods in Consumer

MethodName type Description
accept(T t) abstract Process the given input data and return nothing.
andThen(Consumer<? super T> after) default Chain multiple Consumer function and execute one after the other.

Example and explanation

  • accept(T t)

T is the input given to the consumer function. In below example , we are iterating the List of String in the Lambda function.
Consumer> lenOfString = stringLst -> stringLst.stream().forEach(s -> System.out.println("Length of String ->" + s.length()));
lenOfString.accept(stringLst);

  • andThen(Consumer<? super T> after)

This method takes another consumer as input and execute one consumer after other. In the below example,we are printing the List of string and filtering the element by size. 
Consumer> lenOfString = stringLst -> stringLst.stream().forEach(s -> System.out.println("Length of String ->" + s.length()));
		
Consumer> lenCheck = lengthCheckLst -> lengthCheckLst.stream().filter(s -> s.length() > 5).forEach(s -> System.out.println("Filter condition ->" + s));
				
lenOfString.andThen(lenCheck).accept(stringLst);

Complete Code For Consumer

	
package com.weblearner.functionallambda;

import java.util.ArrayList;
import java.util.List;
import java.util.function.Consumer;

public class ConsumerFunctionalInterface {

	public static void main(String[] args) {

		System.out.println("Example for accept");

		Consumer> lenOfString = stringLst -> stringLst.stream()
				.forEach(s -> System.out.println("Length of String ->" + s.length()));
		
		Consumer> lenCheck = lengthCheckLst -> lengthCheckLst.stream().filter(s -> s.length() > 5)
				.forEach(s -> System.out.println("Filter condition ->" + s));

		List stringLst = new ArrayList<>();
		stringLst.add("Java");
		stringLst.add("PHP");
		stringLst.add("Angular");

		lenOfString.accept(stringLst);

		System.out.println("Example for andThen");		

		lenOfString.andThen(lenCheck).accept(stringLst);

	}

}
