Java8 - Features - Default and Static Method Interface

In this tutorial, we are going to see use of Default and Static method in Interface which was introduced as a new feature in Java8
Interface contains only public and abstract method before Java 8. In Java8, default method is introduced to avoid the need of changing the all the implementation class if we need to add new method in an interface.Consider a example where a Interface is implemented by 100 classes and you need to add a method to it, then we need to override the new method in all 100 classes. Default method is not a replacement for abstract class in java. We will see the behavior of default and static method in interface with few examples.Also we will see how to resolve multiple inheritance with default method.

Interface with Default and Static Method

Created a simple interface with abstract,default and static method. Abstract method need to be implemented in the class 
	public interface DefaultAndStaticInterface {

	public void interfaceMethod();

	default String defaultMethod() {
		return "Default Method in Interface Class";
	}
	
	public static String staticMethod() {
		return "Static Method in Interface Class";
	}

}

Implementation Class

Implementation class overrides the abstract method interfaceMethod(). In the main() , we will invoke all the 3 methods. 
	public class DefaultStaticInterfaceExample implements DefaultAndStaticInterface {
	
	@Override
	public void interfaceMethod() {
		System.out.println("Overrided method of interface");

	}

	public static void main(String args[]) {

		DefaultAndStaticInterface obj = new DefaultStaticInterfaceExample();
		
		// invoking default method
		System.out.println(obj.defaultMethod());
		
		// invoking static method
		System.out.println(DefaultAndStaticInterface.staticMethod());
		
		// invoking interface abstract/overrided method
		obj.interfaceMethod();
	}
}

Output

				
Default Method in Interface Class
Static Method in Interface Class
Overrided method of interface

Multiple Inheritance Problem With Default Method and Solution

Since a class can implement more than one interface, we might face issue if both interface have common default method. Compiler will throw below error.

To resolve this issue, we can implement override the default method in implementation class.Please see the example below.

Two Interface With Same Default Method

				
public interface MultipleInterface_1 {

	default void method1() {
		System.out.println("MultipleInterface_1 method");
	}

}

public interface MultipleInterface_2 {

	default void method1() {
		System.out.println("MultipleInterface_2 method");
	}
}

Solution

				
public class MultipleInheritanceProblem implements MultipleInterface_1, MultipleInterface_2 {

	public void method1() {
		System.out.println("MultipleInheritanceProblem method1");
	}

	public static void main(String args[]) {
		MultipleInheritanceProblem obj = new MultipleInheritanceProblem();
		obj.method1();
	}
}
