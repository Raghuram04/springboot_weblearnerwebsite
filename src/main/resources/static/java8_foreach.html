Java8 Features - forEach()

In this tutorial, we are going to see the forEach() method which is introduced in Java 8 for iterating Collection. We will compare with traditional method and see where to use forEach()
We can iterate any class which implements Consumer Interface using forEach() . Consumer Interface is a Functional Interface which has one abstract method called accept(T t) which returns no result 
@FunctionalInterface
public interface Consumer {
    void accept(T t);
}

Traditional Way of Iterating

Before Java8, we can iterate a Collection using for Enhanced for-loop 
				
	List lstObject = new ArrayList<>();
	lstObject.add("SAM");
	lstObject.add("TOM");
	lstObject.add("BOB");
 		
        for (String val : lstObject) {
			System.out.println("List Value ->" + val);
		}

Example for forEach() with Map and List

The below example shows the way to iterate Map and List using forEach(). You can see, we have implemented the System.out.println() using Lamda to print all the elements in the List and Map 
				
	public static void main(String[] args) {
		Map mapObject = new HashMap<>();
		mapObject.put("A", "SAM");
		mapObject.put("B", "TOM");
		mapObject.put("C", "BOB");

		List lstObject = new ArrayList<>();
		lstObject.add("SAM");
		lstObject.add("TOM");
		lstObject.add("BOB");

       // Iterate Map using forEach()
       
		mapObject.forEach((key, value) -> {
			System.out.println("Key ->" + key + "  ---  " + "Value ->" + value);
		});
	
		// Java foreach for List

		lstObject.forEach(val -> System.out.println("List Value ->" + val));

	}

Result

				
Java8 forEach() to iterate Map
Key ->A  ---  Value ->SAM
Key ->B  ---  Value ->TOM
Key ->C  ---  Value ->BOB
Java8 forEach() to iterate List
List Value ->SAM
List Value ->TOM
List Value ->BOB
The main advantage of using forEach() in Java 8 is we can use along with Stream which support parallel processing . Also it is more readable with less lines of code since we can use Lambda.
