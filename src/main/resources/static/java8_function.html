Java8 Features - Function Interface

In this tutorial, We are going to learn about Function Functional Interface which was introduced as part of Java 8. As learned in previous tutorial about Functional Interface, we have only one abstract method in Predicate Function Interface called apply(Object).

Methods in Predicate

MethodName type Description
apply(T t) abstract Apply the function to give input
andThen(Function<? super R,? extends V> after) default Chain multiple function together. Apply the first function to given input and second function on the evaluated response.

Example and explanation

  • apply(T t)

Interface Function<T ,R> where T is the input and R is the return type.In the below example, we are passing a List and return a List. We are just returing top two elements from the List. 
Function, List> topLang = stringLst -> stringLst.subList(0, 2);
topLang.apply(langLst)

  • andThen(Function<? super R,? extends V> after)

This method takes Function as input and combine two functions together.First function get the top two elements and next one will count the size of array.
List langLst = new ArrayList<>();
langLst.add("Java");
langLst.add("C");
langLst.add("Pascal");
langLst.add("COBOL");

Function, List> topLang = stringLst -> stringLst.subList(0, 2);
Function, Integer> sizeLst = stringLst -> stringLst.size();	

topLang.andThen(sizeLst).apply(langLst);

Complete Code For Function

	
package com.weblearner.functionallambda;

import java.util.ArrayList;
import java.util.List;
import java.util.function.Function;

public class FunctionFunctionalInterface {

	public static void main(String args[]) {
		
		List langLst = new ArrayList<>();
		langLst.add("Java");
		langLst.add("C");
		langLst.add("Pascal");
		langLst.add("COBOL");

		Function, List> topLang = stringLst -> stringLst.subList(0, 2);
		Function, Integer> sizeLst = stringLst -> stringLst.size();	

		System.out.println("Size of List ->" + topLang.andThen(sizeLst).apply(langLst));

	}

}
