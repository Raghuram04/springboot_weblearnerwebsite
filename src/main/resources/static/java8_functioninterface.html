Java8 Features - FunctionInterface and Lambda Expression

In this tutorial, we are going to see the use of FunctionalInterface and Lambda Expression introduced in Java 8. Interface with one abstract method is called Function Interface. Example Callable,Runnable etc. We can define our own interface as Function Interface using @FunctionalInterface annotation. Lambda is an anonymous function without name and doesn't belong to any class.
Before we proceed with actual example to use Lambda and Functional Interface, We will see how to write lambda expression and compare with traditional method. In below example we have add(int a , int b) method returning integer value.We can remove name of the method and return keyword. Since we have only one statement we dont need curly bracket too. 
	// traditional method
	
	public int add(int a,int b) {
		return a+b;
	}
	
	//equivalent Lambda expession for above method

	(a,b) -> a+b;

Lambda Expression without Argument

We have created a simple functional Interface FunctionalInterfaceExample with one abstract method as below. 
				
@FunctionalInterface
public interface FunctionalInterfaceExample {	
	public void test();
}
FunctionalInterfaceNoArgExample class defines the lambda expression for this FuntionInterface without argument. As this method doesn't return anything, we are having a print statement. 
public class FunctionalInterfaceNoArgExample {

	public static void main(String[] args) {

		FunctionalInterfaceExample withoutArg = () -> System.out
				.println("Example of functional interface without argument");
		
		withoutArg.test();
		
	}
}

Output

Example of functional interface without argument

Lambda Expression with Argument

Another FunctionInterface class is created to demonstrate argument Lambda Expression as below. 
				
public interface FunctionalInterfaceExample2 {	
	public int add(int a,int b);
}
FunctionalInterfaceArgumentExample class defines the lambda expression for this FuntionInterface with argument. In this case, we are adding two integer and returning a integer. 
public class FunctionalInterfaceArgumentExample {

	public static void main(String[] args) {

		FunctionalInterfaceExample2 withArg = (a, b) -> a + b;
		System.out.println("Functional Interface with argument  ->" + withArg.add(5, 3));	
		
	}
}

Output

Functional Interface with argument ->8

Lambda Expression For Runnable

Since Runnable Interface is a FunctionInterface, we can use Lambda expression to define the run() method. 
public class FunctionalInterfaceThreadExample {

	public static void main(String[] args) {
		Runnable t = () -> System.out
				.println("Thread example. Runnable is a functional interface with only run() abstract method");

		new Thread(t).start();		
	}
}

Output

Thread example. Runnable is a functional interface with only run() abstract method
Lambda expression helps to minimize the lines of code and more readable. We can define the functionality in any place without the need for any class definition
