Java8 Features - Predicate Functional Interface

In this tutorial, We are going to learn about Predicate Functional Interface which was introduced as part of Java 8. As learned in previous tutorial about Functional Interface, we have only one abstract method in Predicate Function Interface called test().As name says ,this function evaluate an operation to return boolean result.

Methods in Predicate

MethodName type Description
test(T t) abstract Evaluate the predicate and return boolean value
and(Predicate < ? super T > otherPredicate) default Chain multiple predicate function and apply LOGICAL AND condition. Check both predicate function return true.
or(Predicate < ? super T > otherPredicate) default Chain multiple predicate function and apply LOGICAL OR condition. Check at least one predicate function return true.
negate() default Negate the actual response evaluate by predicate.

Example and explanation

  • test(T t)

T is the input given to the predicate function. In below example , we are checking whether given string length is greater than 0. Only abstract method which take T type input, in our case its a string. We have a return a lambda function to override the test method. 
Predicate passwordCheck = pass -> pass.length() > 0;
passwordCheck.test("Weblearner");

  • and(Predicate < ? super T > otherPredicate)

This method takes another predicate as input and evaluate both the predicate and check whether both returns true. we need to call test() to pass input to this function. This is a default method in Predicate Interface,so we are not breaking the rule of Function Interface(one abstract method) 
Predicate passwordCheck = pass -> pass.length() > 0;
Predicate maxLength = pass -> pass.length() > 10;
maxLength.and(passwordCheck).test("Weblearner10");

  • or(Predicate < ? super T > otherPredicate)

This method takes another predicate as input and evaluate either one of the predicate returns true. we need to call test() to pass input to this function. 
Predicate passwordCheck = pass -> pass.length() > 0;
Predicate maxLength = pass -> pass.length() > 10;
maxLength.or(passwordCheck).test("Weblearn");

  • negate()

This method negate the value returned by the test(). 
Predicate passwordCheck = pass -> pass.length() > 0;
Predicate maxLength = pass -> pass.length() > 10;
maxLength.or(passwordCheck).negate().test("Weblearner");

Complete Code For Predicate

	
package com.weblearner.functionallambda;

import java.util.function.Predicate;

public class PredicateFunctionInterface {

	public static void main(String[] args) {

		System.out.println("Test Condition");

		Predicate passwordCheck = pass -> pass.length() > 0;

		System.out.println("Not Empty Check -->" + passwordCheck.test("Weblearner"));
		System.out.println("Not Empty Check -->" + passwordCheck.test(""));

		System.out.println("And Condition");

		Predicate maxLength = pass -> pass.length() > 10;

		System.out.println("Empty and Max length Check -->" + maxLength.and(passwordCheck).test("Weblearner10"));
		System.out.println("Empty and Max length Check -->" + maxLength.and(passwordCheck).test("Weblearner"));

		System.out.println("OR Condition");

		System.out.println("OR Check -->" + maxLength.or(passwordCheck).test("Weblearner"));

		System.out.println("Negate Condition");

		System.out.println("Negate Check -->" + maxLength.or(passwordCheck).negate().test("Weblearner"));

		String response = passingPredicate(10, i -> i > 18);

		System.out.println("Passing Predicate as parameter  -->" + response);

	}

	private static String passingPredicate(int a, Predicate ageLimitCheck) {

		if (ageLimitCheck.test(a)) {
			return "Age limit Check Passed";
		} else {
			return "Age limit Check Failed";
		}

	}

}
