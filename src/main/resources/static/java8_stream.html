Java8 Features - Stream

In this tutorial, We are going to see the Stream Interface introduced in Java8. We will cover the useful methods available with this interface.
Stream is not a data structure, its a interface used to process collection of objects like Arrays,Collection (List,Set etc),Java NIO Operation. Stream provide pipeline options to build our operation on the top of another operation Resultset.

Types of Operations

  • Intermediate Operation
    Returns a modified Stream Object after some operation. We can pipeline on top of the modified Stream object . We keep on chaining on the modified response till we get desired output. Example of Intermediate Operators are map(),filter(),sorted()
  • Terminal Operation
    Returns the final response. No more chaining after the terminal method. We can have only one terminal method on the given operation.Example of Terminal Operation are collect,forEach,reduce

Common Methods

  • count(): Returns the count of element in the given collection
  • distinct(): Returns distinct element in the given collection.
  • parallelStream(): Helps to parallel process the elements in the given collection. We can use this when the order of element processing can be random.
  • filter(): Returns a modified Stream based on the given filter condition.
  • map(): Returns a modified Stream based on the given map condition.Mostly returns the same Stream with modified value.
  • match(): Returns a boolean value. We have two methods anyMatch() and allMatch() returning boolean value based on the condition.
  • reduce(): Reduce the given Stream into one value.
  • collect(): Collect the results of intermediate operation. Mostly used with Collectors class to collect the value in particular format like toList(),toSet()

Example Program with all the operations.

	
public class StreamExample {

	public static void main(String[] args) {
		// TODO Auto-generated method stub

		List stringLst = new ArrayList<>();
		stringLst.add("Sam");
		stringLst.add("Tom");
		stringLst.add("Peter");
		stringLst.add("Peter");

		List intLst = new ArrayList<>();
		intLst.add(1);
		intLst.add(10);
		intLst.add(8);
		intLst.add(5);

		// Array example
		
		System.out.println("Array Example -->");

		String[] arr = new String[] { "Sam", "Tom", "Peter" };
		Stream arrStream = Arrays.stream(arr);
		arrStream.forEach(s -> System.out.println(s));

		// count example

		System.out.println("Count Example -->" + stringLst.stream().count());

		// distinct example

		System.out.println("distinct example -->" + stringLst.stream().distinct().count());

		// parallel Stream example

		System.out.println("parallelStream example");

		stringLst.parallelStream().forEach(s -> System.out.println(s));

		// filter example

		System.out.println("filter example");

		stringLst.stream().filter(s -> s.startsWith("P")).forEach(s -> System.out.println(s));

		// map

		System.out.println("map example");

		Stream mapExampleStream = intLst.stream().map(i -> i * 10);
		mapExampleStream.forEach(i -> System.out.println(i));

		// match

		System.out.println("Any Match example -->  " + stringLst.stream().anyMatch(s -> s.startsWith("P")));
		System.out.println("All Match example -->  " + stringLst.stream().allMatch(s -> s.startsWith("P")));

		// reduce

		System.out.println("Reduce Example -->" + intLst.stream().reduce(0, (i, j) -> i + j));

		// collect
		
		System.out.println("collect example");

		List collectExample = stringLst.stream().map(s -> s.toLowerCase()).collect(Collectors.toList());
		collectExample.parallelStream().forEach(s -> System.out.println(s));

	}

}
