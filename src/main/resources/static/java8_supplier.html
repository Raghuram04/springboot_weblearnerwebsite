Java8 Features - Supplier Functional Interface

In this tutorial, We are going to learn about Supplier Functional Interface which was introduced as part of Java 8. As learned in previous tutorial about Functional Interface, we have only one abstract method in Predicate Function Interface called get().This method will get the value of the Supplier Interface.

Methods in Supplier

MethodName type Description
get() abstract Get the value of Supplier Function Interface.

Example and explanation

  • get()

Interface Supplier<T> will take no argument as input and return T type response. In the below example, we are calculating pie value and returning a Integer value.
Supplier pieVal = () -> { int pie = 22/7;return pie;};
System.out.println("Supplier Interface -->" + pieVal.get());

Complete Code For Consumer

	
package com.weblearner.functionallambda;

import java.util.function.Supplier;

public class SupplierFunctionalInterface {

	public static void main(String[] args) {

		Supplier pieVal = () -> { int pie = 22/7;return pie;};
		System.out.println("Supplier Interface -->" + pieVal.get());

	}

}
