Java8 Features - Time API

In this tutorial, We are going to see the Time API introduced in Java8. We will cover the useful methods available with this interface.
New Time API was introduced in Java 8 to reduce the complexity in handling Date and Calendar features. In the new API, we have LocalDate,LocalTime,Period,Duration classes with many simple method to do date manipulation.We will show the example of all the class.

New Date Classes

  • LocalDate : Class have useful methods to get the Current date and extract individual value.
  • LocalTime : Class have useful methods to get the Current time and extract individual value.
  • LocalDateTime : Class have useful methods to get the Current date and time and extract individual value.
  • Period : Class is generally used to get the difference between two dates.
  • Duration : Class is generally used to get the difference between two times.

Example for LocalDate

We can get the current date value by using LocalDate.now() or create a LocalDate object using LocalDate.of(YEAR, MONTH, DAY) 
		LocalDate localDate = LocalDate.now();

		System.out.println("Current Local Date --> " + localDate);
		System.out.println("Current Month --> " + localDate.getMonthValue());
		System.out.println("Current Day --> " + localDate.getDayOfMonth());
		System.out.println("Current Year --> " + localDate.getYear());

		LocalDate preDefinedDate = LocalDate.of(2021, 10, 31);

		// Comparing date . Date are same if value is zero
		System.out.println(preDefinedDate.compareTo(localDate));

Example for LocalTime

We can get the current time value by using LocalTime.now() or create a LocalTime object using LocalTime.of(HOUR, MINUTE, SECOND, NANOSEC) 
		LocalTime time = LocalTime.now();

		System.out.println("Local Time -->" + time);
		System.out.println("Hour of current time -->" + time.getHour());
		System.out.println("Minute of current time -->" + time.getMinute());
		System.out.println("Seconds of current time -->" + time.getSecond());
		System.out.println("NanoSeconds of current time -->" + time.getNano());

		LocalTime preDefinedTime = LocalTime.of(9, 40, 40, 0);

		System.out.println(preDefinedTime.compareTo(time));

Example for LocalTimeDate

We can get the current date and time value by using LocalDateTime.now() 
              LocalDateTime dateTime = LocalDateTime.now();

Example for Period

Period class is used to find difference between two dates.It have static methods to perform many actions like between,from,ofDays,ofMonths etc 
				Period.between(preDefinedDate, localDate)

Example for Duration

Duration class is used to find difference between two times.It have static methods to perform many actions like between,from,ofDays,ofMonths etc 
				Duration.between(time,addedTime)

Example for ChronoUnit

ChronoUnit is a ENUM introduced in java8.DAY,MONTH,YEAR,HOURS,MINUTE etc.We can use this ENUM for date manipulation. 
				dateTime.plus(2, ChronoUnit.MONTHS)
				dateTime.plus(2, ChronoUnit.YEARS)

Complete Code

package com.weblearner.timeapi;

import java.time.Duration;
import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.LocalTime;
import java.time.Period;
import java.time.temporal.ChronoUnit;

public class TimeAPIExample {

	public static void main(String args[]) {

		// LocalDate example

		LocalDate localDate = LocalDate.now();

		System.out.println("Current Local Date --> " + localDate);
		System.out.println("Current Month --> " + localDate.getMonthValue());
		System.out.println("Current Day --> " + localDate.getDayOfMonth());
		System.out.println("Current Year --> " + localDate.getYear());

		LocalDate preDefinedDate = LocalDate.of(2021, 10, 31);

		// Comparing date . Date are same if value is zero
		System.out.println(preDefinedDate.compareTo(localDate));

		LocalTime time = LocalTime.now();

		System.out.println("Local Time -->" + time);
		System.out.println("Hour of current time -->" + time.getHour());
		System.out.println("Minute of current time -->" + time.getMinute());
		System.out.println("Seconds of current time -->" + time.getSecond());
		System.out.println("NanoSeconds of current time -->" + time.getNano());

		LocalTime preDefinedTime = LocalTime.of(9, 40, 40, 0);

		System.out.println(preDefinedTime.compareTo(time));

		LocalDateTime dateTime = LocalDateTime.now();

		System.out.println("LocalDateTime --> " + dateTime);

		// ChronoUnit example. ChronoUnit is a ENUM introduced in java 8.
		// DAY,MONTH,YEAR,HOURS,MINUTE etc..

		System.out.println("Adding month to current date using Chronounit  --> " + dateTime.plus(2, ChronoUnit.MONTHS));

		// Period and Duration
		System.out.println("Period between two dates -->" + Period.between(preDefinedDate, localDate));
		LocalTime addedTime = time.plus(1, ChronoUnit.HOURS);
		System.out.println("Duration between two times -->" + Duration.between(time,addedTime));

	}

}
