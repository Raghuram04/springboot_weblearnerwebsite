Spring Boot - Generating Random Number

Overview

Springboot provides RandomValuePropertySource class to generate random number in spring application. It return a random value for any property with "random" in it. We also have option to specify INT,LONG type of random value with range level. It also supports generating UUID using java.util.UUID.

About Demo

In this Demo,We are going to show how to generate random value in spring boot application.

Prerequisites

  • Maven 3.3.9
  • Java 8
  • Eclipse

pom.xml

Link to source code is available at the bottom of the page

RandomGenerator

This class will generate random number for below fields. We are using random value generator
  • random.int
  • random.long
  • random.uuid
package com.weblearner.springboot;

import java.util.UUID;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.boot.context.properties.ConfigurationProperties;
import org.springframework.stereotype.Component;

@Component
@ConfigurationProperties
public class RandomGenerator {

	@Value("${random.int[1,10000]}")
	private int id;
	@Value("${random.long(1000000)}")
	private Long longId;
	@Value("${random.uuid}")
	private UUID uuid;
	
	// setter and getter
}

SpringbootRandomApplication

SpringbootRandomApplication class is used to start the application and we are getting the bean object of RandomGenerator class to display the random value.
package com.weblearner.springboot;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;
import org.springframework.context.ConfigurableApplicationContext;

@SpringBootApplication
public class SpringbootRandomApplication {

	public static void main(String[] args) {
		ConfigurableApplicationContext context = SpringApplication.run(SpringbootRandomApplication.class, args);
		RandomGenerator bean = context.getBean(RandomGenerator.class);
		System.out.println("Integer value" + bean.getId());
		System.out.println("Long Value" + bean.getLongId());
		System.out.println("Random UUID" + bean.getUuid());
	}

}
