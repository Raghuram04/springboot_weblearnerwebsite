Spring Boot - Spring MVC Example

Overview

Spring MVC is build around DispatcherServlet which will redirect the request to corresponding handler to execute the business logic. @Controller defines the Controller class which will accept the request. @RequestMapping defines the request URI for particular resource in back end.@ModelAttribute helps to receive the submitting form value in the back end.ModelAndView helps us to bind the model object with view. These are basic element required develop as Spring MVC application.

About Demo

In this tutorial we are going to demonstrate Spring MVC using Spring boot. We have created a simple registration form and submit the request to back end controller.

Controller

Controller class will receive the request based on the URI. @Controller annotation helps the Spring to identify the Controller class. We have two @RequestMapping annotation, one at the class level and another one at method level. This will define the URI path of the request.When user hits /weblearner/home in browser, getHomePage() will be called and ModelAndView object will couple the view(registration.jsp) and model(User) object. 
@Controller
@RequestMapping("/weblearner")
public class HomeController {
	
	@RequestMapping(value = "/home", method = RequestMethod.GET)
	public ModelAndView getHomePage() {
		ModelAndView mv = new ModelAndView();
		User user = new User();
		mv.setViewName("registration");
		mv.addObject("user", user);
		return mv;
	}

	@RequestMapping(value = "/registration", method = RequestMethod.POST)
	public ModelAndView registrationForm(@ModelAttribute("user") User user) {
		ModelAndView mv = new ModelAndView();
		System.out.println("FirstName -->" + user.getFirstName());
		mv.addObject("status", "Successfully Registered");
		mv.setViewName("status");
		return mv;
	}

}

Views

Views are simple jsp page with FORM to get the user data. ModelAttribute in the FORM tag maps with User class.Make sure to use same name as class attribute and path parameter.
